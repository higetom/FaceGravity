# FaceGravity

**「姿勢を変えたら、顔が変わった。」**

顔のたるみ改善のための世界初「姿勢×顔たるみ」統合アプリ

## 起動方法

```bash
cd ~/Desktop/FaceGravity
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。
スマホ表示で確認する場合は、ブラウザのDevToolsでモバイルビューに切り替えてください。

## 技術スタック

- React 18 + Vite
- Tailwind CSS
- Framer Motion（アニメーション）
- Recharts（グラフ）
- Lucide React（アイコン）
- React Router（ルーティング）
